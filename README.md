# AUIFlow
AUIFlow 是基于大型语言模型（LLMs）开发的动态GUI生成系统，旨在通过对话内容自动生成个性化图形用户界面，优化聊天界面的交互体验。该项目通过生成交互式HTML界面，减少用户认知负担，提高任务完成效率，为对话式AI系统带来更智能、更流畅的交互方式。

随着对话式AI系统在各领域的广泛应用，传统纯文本交互模式在处理复杂任务时逐渐暴露出效率低、认知负荷高等问题。用户需要在多轮对话中反复输入指令，而系统无法提供直观的可视化操作界面，导致交互体验不佳。
本课题旨在探索如何利用大型语言模型（LLM）动态生成图形用户界面（GUI），以增强聊天系统的交互能力，提升用户任务完成效率，具体设计内容如下：
1. 动态GUI生成架构设计：研究基于LLM的对话理解与界面生成机制，设计从自然语言对话到GUI组件的映射框架。该架构需支持实时分析对话上下文，识别用户意图，并自动生成对应的界面元素（如表单、按钮、数据可视化组件等），实现"对话即界面"的交互范式。
2. LLM驱动的界面生成方法：研究利用大型语言模型（如GPT系列）生成可执行的HTML/CSS/JS代码的技术路径。包括：设计有效的提示工程策略，使LLM能够输出符合语义的界面代码；构建可复用的UI组件库模板；实现生成代码的安全性过滤与实时渲染。重点解决代码生成的准确性、界面美观度和交互逻辑完整性问题。
3. 交互增强机制与评估：设计用户交互反馈机制，支持用户在生成的GUI上直接操作并反馈至对话系统。构建对比实验平台，通过用户研究评估动态GUI在降低认知负荷、缩短任务完成时间、提升满意度等方面的效果，验证该方法相较于传统文本交互的优势。
